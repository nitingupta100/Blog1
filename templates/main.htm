React

React is a javascript library. The basic philosophy is to divide the HTML page into components. Each component will have its html, javascipt and css files. The data is passed to the component using the props property and each component can have its own event handler. These individual components are encapsulated similar to objects in server side langauages and this makes them easier to reuse in another application with some changes
